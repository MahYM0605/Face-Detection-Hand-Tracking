# Face & Hand Detection using OpenCV and MediaPipe

This project performs real-time face detection and hand tracking using a webcam using Python.

## Features
- Real-time face detection with confidence score
- Hand landmark detection (21 key points)
- Fancy bounding box drawing
- FPS counter

## Technologies Used
- Python
- OpenCV
- MediaPipe

## How to Run
1. Install dependencies:
pip install opencv-python mediapipe (can skip this if you had already installed)

2. Run the program:
python main.py

3. Press Q to quit
